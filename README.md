# 🤖 Claude Code マスター講座

---

## 📖 この講座について

**Claude Code** は、Anthropic 社が提供する CLI（コマンドライン）ベースの AI エージェントです。
本講座では、ブラウザでのコピペ作業を卒業し、ターミナル上で AI と共に本格的な開発ができるようになることを目指します。

### 最終ゴール
> Claude Code を独力で使いこなし、**設計・実装・テスト・デバッグのサイクルを AI と共に高速に回せるようになること**

## ⚡ クイックスタート

### ステップ1：前提環境の確認

```bash
# Node.js のバージョン確認（18以上が必要）
node -v

# Git の確認
git --version

# 環境確認スクリプトを実行（自動チェック）
bash scripts/check-env.sh
```

### ステップ2：Claude Code のインストール

```bash
npm install -g @anthropic-ai/claude-code
claude --version
```

### ステップ3：講座を開始する

```bash
# このディレクトリで Claude Code を起動
claude

# 最初にこう話しかけてみよう
> このプロジェクトの概要を教えて。どの章から始めればいい？
```

## 💡 学習のコツ

1. **必ずターミナルを開いて手を動かす** — 読むだけでは身につきません
2. **失敗を恐れない** — Git があれば何でも元に戻せます
3. **エラーが出たら Claude に丸投げする** — それが AIDD の基本です
4. **各章末の演習を必ずやる** — `docs/exercises/` を確認しましょう
