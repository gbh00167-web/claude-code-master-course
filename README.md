# Claude Code マスター講座（個人事業主版）

> 繰り返しの業務をAIに任せて、あなたの時間を本来の仕事に使う

---

## この講座について

**Claude Code** は、Anthropic 社が提供するAIツールです。
Web版のClaudeと違い、**あなたのMac上のファイルを直接読み書きできる**のが最大の特徴です。

本講座は、**個人事業主の方**が Claude Code を使って日常業務を自動化できるようになることを目指した実践ガイドです。

### 最終ゴール
> 自分の繰り返し業務をClaude Codeで自動化し、毎週・毎月の手作業を減らせること

---

## 対象読者

- Mac を使っている個人事業主（デザイナー・コンサルタント・ライター・カメラマン・士業など職種問わず）
- ターミナルでフォルダの移動（`cd` コマンド）ができる方
- Web版Claudeはある程度使っているが、もっと自動化したい方

## クイックスタート

### ステップ1：環境確認

```bash
# Claude Code をインストール
npm install -g @anthropic-ai/claude-code

# バージョン確認（数字が表示されればOK）
claude --version
```

### ステップ2：講座サイトを開く

```bash
# 必要なライブラリをインストール
npm install

# ブラウザで講座サイトを表示
npm run docs:dev
```

ブラウザで `http://localhost:5173` を開くと講座サイトが表示されます。

### ステップ3：前書きから読む

```
docs/chapters/00-preface.md
```
