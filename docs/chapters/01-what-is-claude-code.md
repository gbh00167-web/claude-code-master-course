## 1.1 Web版Claudeとの「本質的な違い」

まず正直に言います。Web版のClaude（claude.ai）は非常に優秀なAIです。文章を書かせる、アイデアを出させる、質問に答えさせる——これらはすでに十分に活用できているはずです。

では、Claude Code は何が違うのか。

一言でいえば、**「Claude があなたのMacの中で動く」** という点です。

### ブラウザ版は「電話で相談できる顧問」

Web版Claudeは、電話の向こうにいる優秀な顧問のようなものです。相談すれば的確なアドバイスをくれますが、実際にあなたの書類を見るためには「ファックスで送ってもらえますか」という一手間が必要です。返ってきたアドバイスを実行するのは、あなた自身です。

```
あなた → ファイルを開いてブラウザに貼り付け
     → Claudeが回答を生成
あなた → 回答をコピーしてファイルに貼り付け
     → 必要なら再度繰り返す
```

### Claude Code は「自分のオフィスにいる有能なアシスタント」

Claude Code は、あなたのオフィス（Mac）に常駐しているアシスタントです。「先月の議事録フォルダを見て、まだ完了していないタスクをリストアップして」と言えば、アシスタントはフォルダを自分で開いて読み、結果をまとめて報告してくれます。あなたがコピー＆ペーストをする必要はありません。

```
あなた → 「この仕事をして」と一言伝える
Claude Code → 自分でファイルを開き・読み・書き・保存する
あなた → 結果を確認するだけ
```

## 1.2 「ファイルに直接触れる」が意味すること

Claude Code の核心は **「あなたのMac上のファイルとフォルダに直接アクセスできる」** という機能です。これが具体的に何を意味するか、職種別に見てみましょう。

### コンサルタント・士業の方

```
「先月作ったクライアントAへの提案書を参考に、
 同業種のクライアントB向けに内容を調整した提案書を作って。
 会社名・担当者名・金額は後で私が直接入れる。」
```

既存ファイルを読み込んで新しい文書を生成するため、一から書き直す必要がありません。ひな型を維持しながら、文脈に合わせてカスタマイズした下書きが数分で手に入ります。

### ライター・編集者の方

```
「取材メモ（interview-notes.txt）を読んで、
 2000字の記事下書きを作って。
 見出しは3つ、具体的なエピソードは取材メモから引用すること。」
```

取材メモをClaudeが直接読み込んで執筆するため、内容を貼り付ける手間が不要です。「別のトーンで書き直して」「結論を先に持ってきて」という追加指示も、ファイルを読み込み直さずに続けて依頼できます。

### デザイナーの方

```
「clients/フォルダにある全クライアントのフォルダを確認して、
 直近3ヶ月で更新のないプロジェクトフォルダの一覧を作って。」
```

人間がフォルダを一つ一つクリックして確認する作業を、Claudeが自動でこなします。何十個のフォルダがあっても数秒で完了します。

### カメラマン・クリエイターの方

```
「撮影記録フォルダを読んで、今月の撮影件数・合計時間・
 クライアント別の割合を表にまとめて。」
```

複数のテキストファイルやCSVを横断的に読み込んで集計する作業も、ワンアクションで実行できます。

## 1.3 Web版Claudeとの比較一覧

| 機能 | Web版Claude | Claude Code |
|------|:-----------:|:-----------:|
| ファイルを読み込む | △ 手動で貼り付け | ✅ 自動で読み込む |
| ファイルを保存する | ✕ コピーして手動で保存 | ✅ 直接ファイルに書き込む |
| フォルダを横断して処理 | ✕ | ✅ |
| 複数ファイルを一括処理 | ✕ | ✅ |
| 同じ指示を毎回しなくてよい | ✕ | ✅ CLAUDE.md で設定 |
| 外部ツール（Notion等）と連携 | △ 限定的 | ✅ MCP で拡張可能 |
| スマートフォンから使う | ✅ | ✕ Mac のターミナルが必要 |
| 軽い質問・調べ物 | ✅ 手軽 | △ ターミナルを開く手間 |

> **使い分けの目安**
> 「ちょっと聞きたいだけ」→ Web版Claude
> 「ファイルを使った作業」「繰り返し業務の自動化」→ Claude Code

## 1.4 Claude Code が個人事業主に向いている理由

大企業向けのシステムを導入するには、IT部門の承認や複雑なセキュリティ審査が必要です。しかし個人事業主は、自分のMac上で、自分のファイルに対して、今日から使い始めることができます。

初期コストはAnthropicのサブスクリプション費用のみ（詳細は[コスト・料金ガイド](../references/cost-guide.md)参照）。特別なサーバーや開発者の協力は不要です。

そして最も重要なのは、**「自分の業務に合わせて育てられる」** 点です。第6章で学ぶCLAUDE.mdを使えば、あなたの業界用語・仕事のスタイル・よく使うフォーマットをClaudeに記憶させることができます。

---

## 1.5 この章のまとめ

- Claude Code は「AIをMacのターミナルの中に呼び込む」ツール
- Web版との最大の違いは「ファイルに直接触れる」こと
- コピー＆ペーストの往復作業がなくなり、ファイルを使った業務を自動化できる
- 個人事業主は今日から、自分のMacだけで始められる
