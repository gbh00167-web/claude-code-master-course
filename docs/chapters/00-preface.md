## 「AIに頼んでいるのに、なぜ手間が減らないのか？」

個人事業主として仕事をしていると、こんな経験はないでしょうか。

打ち合わせが終わり、録音から書き起こした議事録ファイルを開いて、Web版のClaudeに貼り付けて「要約して」と入力する。出てきた要約を確認して、今度は「アクションリストにして」と追加で入力する。それをコピーしてGoogleドキュメントに貼り付けて、見出しを整えて……。

AIを使っているのに、気づけばコピー＆ペーストの往復作業に追われている。

あるいは、毎月末に売上一覧のExcelを開き、数値を別のファイルにまとめ、請求書テンプレートに金額を入力して……という手順を、今月も先月とまったく同じようにこなしている。

そのもどかしさの正体は、**「AIがあなたのパソコンの外にいる」** ことです。Web版のClaudeやChatGPTは、ブラウザの画面の向こうにいます。あなたがファイルを開いて内容を貼り付けなければ、AIはその存在を知ることができません。

---

## Claude Code が変えること

**Claude Code** は、AIをあなたのMacのターミナル（操作画面）の中に直接呼び込むツールです。

「議事録フォルダを全部読んで、今週のアクションリストを作って」と一言伝えれば、ClaudeはMac上のフォルダを自分で開き、ファイルを読み、まとめた結果を新しいファイルとして保存してくれます。あなたがコピー＆ペーストをする必要はありません。

「先月の提案書を参考に、新しいクライアント向けの提案書の下書きを作って」と頼めば、既存ファイルを読み込んだ上で、新しい文書を生成します。

「このフォルダ内のファイルを全部PDFに変換して」という作業さえ、コマンド一発で任せられます。

## この講座について

本書は、個人事業主の方がClaude Codeを使って日常業務を自動化できるようになるための実践ガイドです。

| 対象 | Mac を使っている個人事業主の方 |
|------|------|
| 前提知識 | ターミナルでフォルダを移動（`cd` コマンド）できる程度 |
| ゴール | 自分の繰り返し業務をClaude Codeで自動化できる |
| 構成 | 前書き + 第1〜8章 |

職種は問いません。デザイナー、コンサルタント、ライター、カメラマン、コーチ、士業……どんな仕事であっても、「繰り返している作業」があればClaude Codeは役に立ちます。

## 読み進め方のアドバイス

**必ずターミナルを開いた状態で読んでください。** 本書は「読むだけ」では効果が出ません。各章の手順を実際に自分のMacで試しながら進めることで、手が覚えます。

「うまくいかない」「エラーが出た」という場面も必ず訪れます。そのときのために、各章に **「ここで詰まったら」** というセクションを設けています。詰まることは失敗ではなく、AIとの付き合い方を学ぶ最良の機会です。
