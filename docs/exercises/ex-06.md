# 第6章 演習 — CLAUDE.md を作る

> **対応テキスト：** `docs/chapters/06-claude-md-settings.md`
> **目安時間：** 30分

---

## この演習のゴール

自分の仕事スタイルを覚えさせる CLAUDE.md を作り、動作を確認する。

---

## 演習6-A：CLAUDE.md を作る

```bash
cd ~/Desktop/claude-practice
claude
```

```
> このフォルダに CLAUDE.md を作ってください。
  内容は以下の通りです：

  # 基本情報
  私は〇〇（自分の職種）です。

  # 文書スタイル
  - 文体：ですます調
  - 箇条書きを積極的に使う
  - 専門用語は使わず平易な日本語で

  # よく使う情報
  - 名前：〇〇（自分の名前）
```

---

## 演習6-B：CLAUDE.md が反映されているか確認する

Claude Code を一度終了して再起動します：

```
/exit
```

```bash
claude
```

再起動後に確認します：

```
> CLAUDE.md に何が書いてありますか？
```

設定した内容が返ってくることを確認してください。

---

## 演習6-C：CLAUDE.md の効果を体感する

```
> 自己紹介文を200字程度で作ってください。
```

CLAUDE.md に書いた名前や職種が使われているか、ですます調になっているかを確認します。

---

## 演習6-D：CLAUDE.md を更新する

「毎回言っていること」を CLAUDE.md に追加します：

```
> CLAUDE.md に以下を追加してください：
  「ファイルを保存するときは、ファイル名に日付（YYYYMMDD形式）を含めること」
```

---

- [ ] CLAUDE.md が作成された
- [ ] 再起動後も設定が引き継がれた
- [ ] CLAUDE.md の内容が回答に反映された
- [ ] CLAUDE.md を更新できた
