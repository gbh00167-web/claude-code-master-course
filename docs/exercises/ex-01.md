# 第1章 演習 — Claude Code とは何かを体感する

> **対応テキスト：** `docs/chapters/01-what-is-claude-code.md`
> **目安時間：** 20分

---

## この演習のゴール

「Claude Code でできること」を自分の業務に当てはめて考えられるようになる。

---

## 演習1-A：「今の自分の業務」を書き出す

テキストエディタ（メモアプリなど）を開いて、以下の質問に答えてください。

```
質問1：あなたが毎週または毎月繰り返している業務を3つ書いてください。

質問2：その中で一番「手間だな」「面倒だな」と感じているのはどれですか？

質問3：その業務で扱っているファイルはありますか？
       （Word、Excel、テキスト、PDFなど）
```

---

## 演習1-B：Web版Claudeと比較してみる

Web版Claude（claude.ai）を開き、以下のプロンプトを入力してみてください：

```
私は〇〇（あなたの職種）です。
毎月「〇〇（繰り返している業務）」をしています。
この業務を効率化するためのアドバイスをください。
```

回答が返ってきたら、こう考えてみてください：

**「この回答を実際に使うためには、あと何ステップの手作業が必要か？」**

- ファイルを開いて内容を貼り付ける手間は？
- 回答を実際のファイルに反映する手間は？

この「残っている手間」をゼロにするのが Claude Code の役割です。

---

## 確認ポイント

- [ ] 自分の業務の中に「ファイルを使った繰り返し作業」がある
- [ ] Web版Claudeで試して「まだ手作業が残る」と感じた
- [ ] 「Claude Code を使うとその手作業が減るかもしれない」とイメージできた
