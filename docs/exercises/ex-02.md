# 第2章 演習 — はじめての起動を体験する

> **対応テキスト：** `docs/chapters/02-first-launch.md`
> **目安時間：** 30分

---

## この演習のゴール

Claude Code をインストールして起動し、基本的な会話ができる状態にする。

---

## 演習2-A：インストール

ターミナルを開いて、以下を順番に実行してください。

```bash
# Claude Code をインストール
npm install -g @anthropic-ai/claude-code

# バージョンを確認（数字が表示されれば成功）
claude --version
```

**うまくいかない場合：** Web版Claude（claude.ai）を開いて以下を貼り付けてください：
```
ターミナルで「npm install -g @anthropic-ai/claude-code」を実行したところ、
以下のエラーが出ました：[エラーメッセージを貼り付け]
解決方法を教えてください。
```

---

## 演習2-B：練習用フォルダを作って起動する

```bash
# デスクトップに練習フォルダを作る
mkdir ~/Desktop/claude-practice

# そのフォルダに移動
cd ~/Desktop/claude-practice

# Claude Code を起動
claude
```

起動できたら挨拶してみましょう：

```
> こんにちは。私は〇〇（あなたの職種）です。よろしくお願いします。
```

---

## 演習2-C：.claudeignore を作る

```
> このフォルダに .claudeignore ファイルを作って。
  .env ファイルと、private/ フォルダ以下をClaudeに読ませないように設定して。
```

確認が来たら `Y` を押します。ファイルが作られたことを確認しましょう。

---

## 演習2-D：終了する

```
/exit
```

---

## 確認ポイント

- [ ] `claude --version` でバージョン番号が表示された
- [ ] `claude` コマンドで起動できた
- [ ] Claudeと日本語で会話できた
- [ ] `/exit` で正常に終了できた
