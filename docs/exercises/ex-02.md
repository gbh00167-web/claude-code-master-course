## 🏋️ 演習2-A：/ask で安全に調査する

```
> /ask このプロジェクトの docs フォルダには何が入っていますか？
  ファイルを変更せず、構成だけ説明してください。
```

📝 **観察ポイント**：`/ask` なしで聞いた場合と挙動は変わりますか？

---

## 🏋️ 演習2-B：.claudeignore を確認・カスタマイズ

```bash
# .claudeignore の内容を確認
cat .claudeignore
```

自分のプロジェクトで除外すべきフォルダを追加してみましょう。

```
> .claudeignore に追加すべきフォルダやファイルがこのプロジェクトにあれば教えて。
  ただしファイルは変更しないで（/ask モード）。
```

---

## 🏋️ 演習2-C：コストを意識した会話設計

5回以上やり取りをした後、以下を試してください：

```
/stats
```

```
/compact
```

`/compact` の前後でトークン数はどう変わりましたか？

## 🏋️ 演習2-D（応用）：意図的に拒否（Refuse）を試す

```
> このプロジェクトの README.md を削除してください。
```

⚠️ Claude が確認を求めてきたら `n`（拒否）を押してみましょう。
拒否したあとの Claude の反応を観察してください。
