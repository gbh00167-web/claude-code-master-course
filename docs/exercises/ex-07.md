# 第7章 演習 — MCP の設定を試す

> **対応テキスト：** `docs/chapters/07-mcp-tools.md`
> **目安時間：** 30〜60分（設定の難しさによる）

---

## この演習のゴール

MCP の概念を理解し、1つの外部ツール連携を試みる。

> **注意：** この章は設定が複雑で、うまくいかないことも多いです。
> 「うまくいかなくて当然」という心構えで気軽に試してみてください。

---

## 演習7-A：MCP の概念を確認する

Claude Code を起動して聞いてみましょう：

```bash
cd ~/Desktop/claude-practice
claude
```

```
> MCP（Model Context Protocol）とは何ですか？
  個人事業主が使えそうな MCP サーバーを3つ教えてください。
  それぞれの設定の難しさも教えてください。
```

どんな連携が可能かを把握します。

---

## 演習7-B：自分の設定状況を確認する

```
> 現在この Claude Code の MCP の設定状況を確認してください。
  どのMCPサーバーが設定されていますか？
```

---

## 演習7-C（任意）：Brave Search MCP を設定してみる

Web検索ができる MCP の設定に挑戦します。

```
> Brave Search MCP を設定したいです。
  API キーの取得方法から設定ファイルへの追記まで、
  手順を順番に教えてください。
  初心者向けに、各ステップで何をしているかも説明してください。
```

Claudeの指示に従って設定を試みてください。

---

## 演習7-D（任意）：設定後に動作確認する

設定に成功したら：

```
> 「2025年の確定申告で変わったポイント」を Web 検索して要約してください。
```

---

## うまくいかなかった場合

設定でエラーが出た場合は：

```
> MCP の設定でエラーが出ました。
  エラーメッセージは：[エラーを貼り付け]
  何が問題か教えて、解決策を教えてください。
```

それでも解決しない場合は、**第7章をスキップして第8章の卒業演習に進んでも構いません。**
