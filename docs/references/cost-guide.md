# コスト・料金ガイド

---

## Claude Code の料金の仕組み

Claude Code の料金には **2種類** あります：

| 種類 | 内容 | 特徴 |
|------|------|------|
| サブスクリプション | Claude Code 本体の月額料金 | 定額制 |
| API 使用料 | Claudeが思考・回答するたびに発生 | 使った分だけ |

> **大切な注意：** Web版Claudeの有料プランとは別の契約です。
> 最新の料金体系は [公式サイト](https://claude.ai/claude-code) で必ず確認してください。

---

## コスト感覚の目安

| 作業の内容 | 大体のコスト |
|----------|-----------|
| テキストファイルを1つ読んで要約する | $0.01〜0.05 程度 |
| 提案書の下書きを1つ作る | $0.05〜0.20 程度 |
| フォルダ内の10ファイルを一括処理 | $0.10〜0.50 程度 |
| 1時間集中して使う | $0.50〜2.00 程度 |

> あくまでも目安です。ファイルの量・長さ・作業の複雑さによって変わります。

---

## 月間上限を設定する（必須）

料金が予想外に高くならないよう、上限を設定しておきましょう：

1. [Anthropic Console](https://console.anthropic.com/) にログイン
2. `Settings` → `Limits` を開く
3. `Monthly Spend Limit` を設定する（最初は $10〜20 がおすすめ）

上限に達すると自動的に止まります。

## コストを抑えるポイント

### コストを確認する習慣

```
/cost
```

現在のセッションのコストが表示されます。

### 長い会話はリセットする

同じセッションで長く話し続けると、過去の会話もすべてClaudeに送られるためコストが増えます。
別の作業に切り替えるときは：

```
/clear
```

で会話履歴をリセットするか、一度 `/exit` してから再起動しましょう。

### 対象ファイルを絞り込む

「全部読んで」より「〇〇フォルダのAとBのファイルを読んで」と絞り込む方がコストを抑えられます。

### .claudeignore で読ませないファイルを除外する

パスワードや機密ファイルは `.claudeignore` で除外すると、コスト削減にもなります。

---

## 想定外のコストが発生したら

1. すぐに `/exit` で Claude Code を終了する
2. [Anthropic Console](https://console.anthropic.com/) でコスト状況を確認する
3. Monthly Spend Limit を一時的に下げる
4. 原因を確認する（大量のファイルを一度に処理しようとしたなど）
